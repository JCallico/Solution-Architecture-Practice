### How Study Groups Work

1. **Group Formation**
   - Identify common interest
   - Form group (3-8 members)
   - Set schedule
   - Define goals

2. **Regular Sessions**
   - Review material
   - Discuss concepts
   - Share insights
   - Practice together

3. **Collaborative Learning**
   - Share resources
   - Explain to each other
   - Quiz each other
   - Support and encourage

4. **Track Progress**
   - Set milestones
   - Celebrate achievements
   - Adjust approach
   - Stay motivated
