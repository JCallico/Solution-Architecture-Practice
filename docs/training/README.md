## Quick Start

### For New Architects
**Week 1-2**: Foundation
1. Read [Practice Overview](./onboarding/practice-overview.md)
2. Complete [Tools Setup](./onboarding/tools-setup.md)
3. Review [Architecture Principles](../practice-overview/principles.md)
4. Shadow senior architect on project

**Week 3-4**: Learning
1. Complete [Architecture Fundamentals](./learning-paths/architecture-fundamentals.md)
2. Study [C4 Model](../frameworks/c4-model.md)
3. Review [Patterns Library](../knowledge-base/patterns/README.md)
4. Attend Architecture Council meeting

**Week 5-8**: Practice
1. Begin first architecture engagement
2. Create first ADR
3. Present at architecture review
4. Contribute to knowledge base

**Week 9-12**: Integration
1. Lead architecture review
2. Mentor junior team member
3. Present tech talk
4. Complete first major deliverable

See [New Architect Onboarding](./onboarding/new-architect-onboarding.md) for complete plan.

### For Engineers/Developers
**Getting Started**:
1. Review [Architecture Principles](../practice-overview/principles.md)
2. Learn [C4 Model](../frameworks/c4-model.md) for documentation
3. Understand [ADR Process](../frameworks/adr-framework.md)
4. Browse [Patterns Library](../knowledge-base/patterns/README.md)

**When Working with Architects**:
- Review [Engagement Model](../processes/engagement-model.md)
- Understand [ARB Process](../processes/arb/README.md)
- Use [Architecture Templates](../templates/README.md)

## Certification Paths

### AWS Certifications
**Recommended Path**:
1. AWS Certified Solutions Architect - Associate (foundational)
2. AWS Certified Solutions Architect - Professional (advanced)
3. Specialty: Security, Advanced Networking, or Data Analytics

**Support Provided**:
- Training budget
- Practice exams
- Study group
- Time to prepare

### Azure Certifications
**Recommended Path**:
1. Azure Fundamentals (AZ-900) (optional)
2. Azure Solutions Architect Expert (AZ-305)
3. Azure Security Technologies (AZ-500) or specialty

### Google Cloud Certifications
**Recommended Path**:
1. Google Cloud Digital Leader (optional)
2. Professional Cloud Architect
3. Professional Cloud Security Engineer or specialty

### Architecture Certifications
- **TOGAF 9 Certified**: Enterprise architecture framework
- **Zachman Certified Architect**: Enterprise architecture framework
- **Open Group Certified Architect**: Multiple levels

### Technology Certifications
- **CKA/CKAD**: Kubernetes administration and development
- **CISSP**: Security architecture (for security architects)
- **CDMP**: Data management (for data architects)

## Training Catalog

### Internal Training

#### Architecture Fundamentals Workshop
- **Duration**: 2 days
- **Frequency**: Quarterly
- **Audience**: New architects, engineers
- **Topics**: Architecture thinking, documentation, patterns
- **Format**: Instructor-led with hands-on exercises

#### C4 Model Deep Dive
- **Duration**: 4 hours
- **Frequency**: Monthly
- **Audience**: All architects and engineers
- **Topics**: C4 levels, notation, tooling, best practices
- **Format**: Workshop with real examples

- **Duration**: 3 days
- **Frequency**: Quarterly
- **Audience**: Cloud architects, engineers
- **Topics**: Cloud services, Well-Architected, IaC, security
- **Format**: Hands-on labs

#### Microservices Design Workshop
- **Duration**: 2 days
- **Frequency**: Bi-annually
- **Audience**: Solution architects, tech leads
- **Topics**: DDD, service decomposition, patterns

- **Frequency**: Monthly
- **Audience**: All architects and leads
